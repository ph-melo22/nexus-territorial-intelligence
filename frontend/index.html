Plataforma de Inteligência Territorial para o Setor Público Brasileiro

Azure AI Agent Service MCP · Model Context Protocol Dados Abertos do Governo Federal
🎯

Por Quê

Municípios brasileiros gerem bilhões em transferências federais com capacidade limitada de cruzar dados fiscais, sociais e territoriais. Decisões são tomadas em silos — sem visibilidade integrada.

⚙️

Como

Um Router Agent (Azure AI Agent Service + GPT-4o) recebe a pergunta, planeja e despacha agentes especializados que consultam APIs governamentais abertas em paralelo. Resultados são cruzados pelo código IBGE e sintetizados em linguagem natural.

📊

O Quê

Plataforma MCP (Model Context Protocol) com 4 ferramentas de inteligência: fiscal, territorial, social e cruzamento multi-dimensional — para qualquer um dos 5.570 municípios brasileiros.
