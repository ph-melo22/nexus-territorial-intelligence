## Resumo em uma linha

NEXUS é uma plataforma multi-agente de inteligência territorial que cruza dados públicos brasileiros (fiscal, territorial, social) via GPT-4o + Azure AI Foundry. Está deployado no Railway, código no GitHub, submetido ao Agents League Hackathon da Microsoft (prazo: 14/06/2026). **Único bloqueante: vídeo demo no YouTube.**
